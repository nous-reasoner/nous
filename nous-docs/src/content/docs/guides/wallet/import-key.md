---
title: Import Private Key
description: Import an external private key into your wallet.
---

You can import standalone private keys (not derived from your mnemonic) into your NOUS wallet.

## How to Import

1. Go to **Wallet** → **Addresses**
2. Click **Import Key**
3. Enter the 64-character hex private key
4. Click **Import**
5. The address appears in your address list tagged as "Imported"

## Important Notes

- Imported keys are **not** derived from your recovery phrase
- If you restore your wallet from the mnemonic, imported keys will **not** be recovered
- Back up imported private keys separately
- Imported addresses work the same as HD addresses for sending and receiving

## Export a Private Key

To export a private key for any address:
1. Go to **Addresses**
2. Click the address to view details
3. The private key is displayed (keep it secret!)
