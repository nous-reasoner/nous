NOUS

Mining = Reasoning. A blockchain where proof of work is proof of thought.

Constitution

I. 1 NOUS per block, forever.

II. Every valid block contains a solution to an NP-complete problem.

III. Any entity that can reason — human or AI — may participate equally.

How It Works

  • Mining requires solving a 3-SAT formula (256 variables, 986 clauses)
  • No GPU, no ASIC — CPU only, 50KB memory
  • ASERT difficulty adjustment every block
  • 150-second block target
  • Cogito Consensus: 3-SAT + SHA-256 + ASERT

The Mining Equation

Three lines. That's all it takes.

Find (seed, S) such that: F = Generate3SAT(prev_hash ‖ seed, n=256, m=986) S satisfies all 986 clauses of F SHA256(header ‖ seed ‖ S) < target

Why 256 variables? Why 986 clauses?

3-SAT undergoes a phase transition at clause-to-variable ratio r ≈ 4.267.

Below this threshold: solutions likely exist.
Above it: almost certainly unsatisfiable.

At r = 3.85 (the critical region just below the threshold):

  • The problem is hardest to solve — NP-complete in practice
  • Solutions are guaranteed to exist
  • No algorithm does significantly better than exhaustive search
986 = ⌈256 × 3.85⌉

Not an arbitrary choice. The math chose it.

Bitcoin vs NOUS

Bitcoin

Find nonce where SHA256(header ‖ nonce) < target

Better hardware → more hashes → more coins

Result: ASIC arms race, energy waste

NOUS

Find (seed, S) where S solves an NP-complete problem

Better algorithm → faster solving → more coins

Result: algorithmic arms race, scientific progress

Quick Start

git clone https://github.com/nous-reasoner/nous.git
cd nous
go build -o nousd ./cmd/nousd/
go build -o nous-cli ./cmd/nous-cli/
./nous-cli createwallet
./nousd --reason --seeds seed1.nouschain.org:8333
